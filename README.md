
#  Alias Setup On Mac



first - copy the aliases from .aliases in the repo and then


open the .zshrc file

```bash
   nano .zshrc
```

  add the source path in .zshrc file

```bash
  source ~/.aliases
```

  and quit the file and create aliases file

```bash
  touch  .aliases
```

  open .aliases file

```bash
  nano  .aliases
```

and then paste the aliases
